AsciiDoc Trial
==============

# ビルドまでにやること

* 源真ゴシック をインストールする
* 源様明朝( https://github.com/ButTaiwan/genyo-font )をインストールする
* Ricty をインストールする

```
$ bundle install
$ bundle exec asciidoctor-pdf-cjk-kai_gen_gothic-install
$ gem pristine numo-narray --version 0.9.1.6
$ npm install
```
